# forkcast
AI-powered weekly meal planning app

## Getting Started

### Prerequisites
- Node 20+
- npm

### Local Development
```bash
cp .env.local.example .env.local   # fill in your keys
npm install
npm run dev
```

### Docker
```bash
cp .env.local.example .env.local   # fill in your keys
docker compose up --build
```
The app will be available at `http://localhost:3000`.
